---
title: Sync
description: Add dotfiles to the local state. If no targets are specified, modify all dotfiles that are currently in the local state so that they match the active state.
---

## Usage

```bash title="Shell"
punkt sync [<options>] [<targets>]
```
## Examples

```bash
punkt sync
punkt sync ~/.txt ~/a
punkt sync -i ".*\.txt"
punkt sync -i ".*\.txt" -x ".*/a\.txt" /Users/a
```

## Options
### Common Options

| Flag | Description |
| --- | --- |
| `-r`, `--recursive` | Sync files recursively <Callout title="True by default" type="warn">Can be disabled with `--recursive=false`</Callout>|

| Option | Description |
| --- | --- |
| `-i`, `--include` regex | Include files matching the given regex |
| `-x`, `--exclude` regex | Exclude files matching the given regex |

<Callout>If both `--include` and `--exclude` are specified, their intersection is used.</Callout>

### Global Options

| Flag | Description |
| --- | --- |
| `-h`, `--help` | Show help message |
