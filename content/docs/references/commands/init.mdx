---
title: Init
description: Initialise a Punkt local repository for storing the clones of the synced dotfiles (local state). If an URI to a remote repository is provided, the remote repository will be cloned as the local state.
---

## Usage

```bash title="Shell"
punkt init [<options>] [<repo>]
```
## Examples

```bash
punkt init
punkt init audrey
punkt init audrey/dotfiles
punkt init --ssh https://github.com/audrey/dotfiles.git
punkt init --ssh --branch main --depth 1 audrey/dotfiles
```

## Options

| Flag | Description |
| --- | --- |
| `--ssh` | Use SSH instead of HTTPS |

### Common Options

| Flag | Description |
| --- | --- |
| `-h`, `--help` | Show help message |
| `-r`, `--recursive` | Sync files recursively <Callout title="True by default" type="warn">Can be disabled with `--recursive=false`</Callout>|

| Option | Description |
| --- | --- |
| `-i`, `--include` regex | Include files matching the given regex |
| `-x`, `--exclude` regex | Exclude files matching the given regex |

<Callout>If both `--include` and `--exclude` are specified, their intersection is used.</Callout>
