## Usage

```bash title="Shell"
punkt list [<options>] [<paths>]
```

## Examples

```bash
punkt list
punkt list ~/.txt ~/audrey
punkt list -i ".*.txt"
punkt list --path-style local-relative -i ".*.txt" -x ".*/a.txt" /users/audrey
```
