---
title: active-path
description: Print the path to each target's active state when their local state path is given. If no targets are specified then print the path to the active state directory.
---

## Usage

```bash title="Shell"
punkt active-path [<options>] [<targets>]
```

## Examples

```bash
punkt active-path
punkt active-path ~/.local/share/punkt/punkt_audrey
```
